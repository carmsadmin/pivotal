pivotal-simple-jpa
==================

Pivotal Simple Java Persistence API



* Pivotal SimpleJPA - ITS VERY SIMPLE AND FAST
* Simple object-relational-mapping library works with most JPA-annotations.

- Current version 1.2 - STABLE
